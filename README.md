# To Use 

## Clone
```
git clone https://github.com/orandorisk/javascript-starter-project-webpack.git
```

## Go To Project
```
cd javascript-starter-project-webpack
```

## Install
```
npm install
```

## Start The Project
```
npm run start-dev
```
